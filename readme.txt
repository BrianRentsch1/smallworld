How to compile on Linux:
    ->type "make" in the command line
    ->compilation should happen automatically

How to run on Linux:
    -> type "./smallWorld <M value>" in the command line
    -> <M value> is the average degree of the graph, and should be entered as a single number

/********************************************************************************************/
Example:

compile:
        make

run:
        ./smallWorld 20
